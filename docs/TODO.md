## 4月4日

完成了初版0.3.0

下一版本计划优化：
1. CI工作流方面，需要将window bundle与linux bundle打开，目前的机制相当于是便携版。我们还需要发布安装版，并且给以明确的明明。
2. 缓存机制优化，当前情况下，已经加载出的图片在滑动下去之后再滑动回来，需要重新渲染，这应该是机制上有问题。我们应该保证至少在短时间内可以不出问题。当然为了避免内存占用，可以设置一个允许保留的张数范围。
3. 图片详情优化，当我们点开一张图片之后，应该可以点击或左右键翻动前后一张图片。并且可以放大缩小。以及应该有一个info按钮用于控制显示图片详情或统计信息。这些按钮如果没有占用到图片显示区域时应该是可见的，如果重叠，则只有鼠标悬停才可见。
4. 快捷键优化，需要上面所述的左右翻动的快捷键，以及点开图片可以通过esc返回。alt+num可以切换标签。
